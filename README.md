## Install

    cd $HOME
    git clone git@github.com:gustaveen/dotfiles.git

## Setup

    source $HOME/dotfiles/setup.sh

## Install Zim
    
    git clone --recursive https://github.com/zimfw/zimfw.git ${ZDOTDIR:-${HOME}}/.zim

## install Brew

    cd ~/dotfiles && brew bundle
